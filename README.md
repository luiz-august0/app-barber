# app-barber

Aplicativo para gerenciamento de agendamentos direcionado a barbearias desenvolvido com NodeJS integrado com banco de dados MySQL, React Native e ReactJS. Neste aplicativo foi utilizada tecnologias como: integração com google maps, redux para gerenciamento de estado da aplicação, async-storage para salvamento e recuperação de dados do usuário, expo-location para localização em tempo real e entre outras.
